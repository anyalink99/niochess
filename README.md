# niochess

Шахматы в реальном времени без очередности ходов. Фигуры не телепортируются — они **летят** с одной клетки на другую за фиксированное время (по умолчанию ~10 секунд, не зависит от дистанции). Пока фигура в полёте — она неуязвима и не бьёт. Взятие происходит **на финише**, когда фигура «доезжает». Кто первым съест короля — тот и победил.

## Как играть

- Кликни свою фигуру, затем клетку из подсвеченных — фигура полетит.
- В полёте фигура неуязвима и не занимает клетку.
- На финише: доехала на врага → взятие; две вражеские на одну клетку → обе гибнут; набег на свою занятую клетку → пришедший пропадает; пешка на край → ферзь.
- Шахов и мата нет — цель в том, чтобы съесть короля.
- ИИ начинает ходить только после нажатия **«Новая партия»**.

## Режимы

- **Одиночная** — белые (ты) против ИИ за чёрных. Поведение ИИ настраивается ползунками (агрессия, снайпинг-перехват, время реакции короля, темп ходов).
- **Сетевая (P2P)** — двое по одному коду комнаты. Сигналинг и соединение через [Trystero](https://github.com/dmotz/trystero) (стратегия nostr, без своего бэкенда). Хост ведёт авторитетную симуляцию, гость шлёт ходы и получает снапшоты.

## Запуск

P2P (динамический импорт модуля + WebRTC) работает по `http(s)`, а не по `file://`. Подними локальный сервер из корня проекта:

```bash
python -m http.server 8123
# затем открой http://localhost:8123/
```

Чтобы сыграть по сети, открой две вкладки (или поделись ссылкой), на вкладке «Сетевая» введи общий код и нажми «Подключиться» с обеих сторон.

## Структура

```
index.html        разметка
css/styles.css    стили (поверх токенов niokit)
js/state.js       константы и общее состояние
js/engine.js      правила, ходы, разрешение прилётов
js/ai.js          ИИ (защита короля, снайпинг, случайные ходы)
js/net.js         P2P через Trystero
js/game.js        старт партии / стартовый экран
js/render.js      построение доски и отрисовка кадра
js/ui.js          ввод, контролы, игровой цикл, инициализация
```

## Зависимости (через CDN)

- [niokit](https://github.com/anyalink99/niokit) — токены дизайна и UI-компоненты.
- [Trystero](https://github.com/dmotz/trystero) — P2P-соединение и сигналинг.
